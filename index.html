Introduction – Why AI agents are exploding in 2025/2026?

Level ★☆☆☆☆

A modern AI agent is no longer just a chatbot. It is an autonomous system that:

  • reasons step by step
  • uses tools (API, calculator, web, code…)
  • keeps long-term memory
  • collaborates with other agents
  • makes complex decisions over multiple steps

Most popular frameworks right now:

  • LangGraph (recommended by LangChain for production)
  • CrewAI (very simple for role-based multi-agent teams)
  • AutoGen (Microsoft – agent conversations)
  • OpenAI Swarm / Agents (lightweight & managed)

Step 1 – First ultra-basic ReAct agent

Level ★★☆☆☆

The ReAct pattern = **Reason** (think) + **Act** (execute) + observe result → loop until final answer.

from openai import OpenAI
client = OpenAI()
def get_weather(city: str) -> str:
    return f"{city}: 24°C, sunny (2026 simulation)"
tools = [{
    "type": "function",
    "function": {
        "name": "get_weather",
        "description": "Get the weather for a city",
        "parameters": {
            "type": "object",
            "properties": {"city": {"type": "string"}},
            "required": ["city"]
        }
    }
}]
question = "What's the weather like in Osasco today?"
messages = [{"role": "user", "content": question}]
response = client.chat.completions.create(
    model="gpt-4o-mini",
    messages=messages,
    tools=tools
)
# ← The real ReAct loop starts here (see step 2)
print(response.choices[0].message.tool_calls)

while True:
    resp = client.chat.completions.create(...)
    if resp.choices[0].message.tool_calls:
        for call in resp.choices[0].message.tool_calls:
            if call.function.name == "get_weather":
                args = json.loads(call.function.arguments)
                result = get_weather(**args)
                messages.append({"role": "tool", "content": result, "tool_call_id": call.id})

Step 4 – Modern control with LangGraph

Level ★★★★☆

LangGraph replaces the while loop with a **state graph** → branches, cycles, human-in-the-loop, persistence, easy debugging.

Minimal example:

from langgraph.graph import StateGraph, START, END
graph = StateGraph(...)
graph.add_node("agent", call_llm)
graph.add_node("tools", execute_tools)
graph.add_conditional_edges("agent", should_continue)
graph.add_edge("tools", "agent")
graph.add_edge(START, "agent")
app = graph.compile()
